# Medical Clinic System

## Project Overview
The Medical Clinic System is a web application designed to manage the operations of a medical clinic. It provides functionalities for patient registration, appointment scheduling, medical records management, and billing.

## Setup Instructions
1. **Clone the repository:**  
   ```bash
   git clone https://github.com/kingebrahimopq-create/medical-clinic-system.git
   cd medical-clinic-system
   ```  
2. **Install dependencies:**  
   ```bash
   npm install
   ```  
3. **Start the server:**  
   ```bash
   npm start
   ```  
4. **Access the application:**  
   Open your web browser and navigate to `http://localhost:3000`.

## Features
- **Patient Management:**  
  Manage patient details and history.
- **Appointment Scheduling:**  
  Schedule and manage appointments with various healthcare providers.
- **Medical Records:**  
  Store and retrieve medical records for patients securely.
- **Billing System:**  
  Handle billing and payments efficiently.

## Tech Stack
- **Frontend:**  
  - React.js  
  - Bootstrap
- **Backend:**  
  - Node.js  
  - Express.js
- **Database:**  
  - MongoDB

## Usage Guides
- **Creating a new patient:**  
  - Navigate to the "Patients" section and click on "Add Patient". Fill in the required details and save.
- **Scheduling an appointment:**  
  - Go to the "Appointments" section and select the time slot. Assign a patient and confirm the appointment.
- **Viewing medical records:**  
  - Select a patient from the patients list and access their medical records.
